# Análise de Sinistros de Trânsito

## Sobre o Projeto

Este projeto tem como objetivo analisar dados de sinistros de trânsito com vítimas,
identificando padrões e tendências que possam contribuir para a melhoria da
segurança viária.

A análise foi realizada utilizando técnicas de análise exploratória de dados
(EDA) e visualização através de dashboard interativo.

---

## Problema

Os acidentes de trânsito representam um problema relevante de segurança pública.
A análise de dados pode auxiliar na identificação de regiões com maior incidência
de acidentes e nos tipos de ocorrências mais frequentes.

Com base nessas informações, é possível apoiar decisões estratégicas voltadas
à prevenção e redução de acidentes.

---

## Fonte de Dados

Os dados utilizados neste projeto são estruturados e foram disponibilizados em
formato de planilha.

Variáveis analisadas:

- Data do acidente
- Tipo de acidente
- Região administrativa
- Existência de feridos

---

## Análise Exploratória de Dados (EDA)

Durante a análise foram realizadas as seguintes etapas:

- Limpeza dos dados
- Remoção de registros duplicados
- Organização das colunas
- Análise de distribuição dos acidentes

---

## Principais Insights

A análise identificou alguns padrões importantes:

- A região Nordeste apresentou o maior número de acidentes.
- O tipo de acidente mais frequente foi abalroamento com vítima.
- A maioria dos registros não apresentou feridos.

Esses resultados podem auxiliar na elaboração de políticas de prevenção
e melhoria da segurança no trânsito.

---

## Dashboard

O dashboard interativo foi desenvolvido utilizando o Google Looker Studio
para facilitar a visualização dos dados e identificação de padrões.

O painel apresenta:

- Total de acidentes
- Acidentes por região
- Tipos de acidente
- Distribuição temporal dos acidentes
